Usage Notes
---------------------------------------------------
- This Application meets all requirements giving in the test assignment

1.	Create a web form which will allow a student to request graduation information. The form must contain the following form fields:
a.	first name
b.	last name
c.	phone
d.	email
2.	All fields are required.
3.	Validate the values entered in the phone and email fields.
4.	The page must be responsive.
5.	The page should have the OCC look and feel.
6.	Add the content provided in the page_content file.
7.	Display an appropriate confirmation after the form has been successfully submitted.
8.	Use the master page and the css file provided.
9.	Use ASP.NET/VB.NET
10. Unit test

To verify extract file and open in Visual Studios
When the Default Landing Page opens, Click "Go To Project" Button.
